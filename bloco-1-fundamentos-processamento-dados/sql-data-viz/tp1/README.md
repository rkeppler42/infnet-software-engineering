# TP1 – Introdução à Visualização de Dados e SQL

Performance Test 1. Introduction to Google Looker Studio: connecting CSV and Google Sheets data sources, creating exploratory visualizations, and organizing a dashboard.

> 📊 This TP consists of a single Looker Studio dashboard connecting two datasets from Kaggle on the topic of video games.

## Dashboard

### 🎮 Video Game Explorer
An exploratory dashboard built from two Kaggle datasets covering game reviews (OpenCritic) and completion times (HowLongToBeat). Includes visualizations from both a CSV source and a Google Sheets source.

- [HowLongToBeat Games Completion Times](https://www.kaggle.com/datasets/kasumil5x/howlongtobeat-games-completion-times) — Google Sheets
