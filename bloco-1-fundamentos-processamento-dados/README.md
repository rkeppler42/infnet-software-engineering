## 🎯 Learning Outcomes

### Introdução à Visualização de Dados e SQL
- Visualize CSV data using Google Looker Studio
- Build interactive reports with Google Looker Studio
- Write SELECT queries in SQL
- Extract insights using GROUP BY and ORDER BY

### Introdução à Programação com Python
- Write simple Python programs using a code playground IDE
- Work with strings, conditionals, loops, and lists
- Create custom functions in Python

### SQL e Modelagem Relacional
- Design relational database models with constraints
- Apply normalization rules to relational models
- Manipulate database objects using SQL
- Perform aggregations across multiple tables

### Python para Processamento de Dados
- Work with advanced string indexing, slicing, and methods
- Handle files, sets, and JSON in Python
- Find and fix bugs in Python programs
- Implement web clients and services in Python

### Projeto de Bloco
- Convert tabular data into dashboards using Google Looker Studio
- Write Python programs of increasing complexity
- Write varied SQL queries across different contexts
