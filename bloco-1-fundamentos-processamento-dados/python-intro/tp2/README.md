# TP2 – Introdução à Programação com Python

Performance Test 2. Covers string manipulation, formatting, and basic input/output operations.

---

## Exercises

### Ex01 – Multiline Strings

Create and store a multiline file header as a single string variable.

### Ex02 – String Concatenation (SKU Generator)

Build a product SKU by concatenating a category prefix and a serial number.

### Ex03 – String Repetition (Separator Line)

Generate a visual separator line by repeating a character a given number of times.

### Ex04 – String Methods (Swap Case)

Capture user input and display the original message alongside its swapped-case version.

### Ex05 – String Formatting (Name Badge)

Format a full name to uppercase and a badge name to title case, then build a security ID string.

### Ex06 – String Search (Password Validator)

Check if a password contains specific special characters and verify if it is all-lowercase or all-uppercase.

### Ex07 – String Length (SMS Validator)

Calculate the total length of an SMS message with prefix and suffix, and determine how many characters it is over or under the 160-character limit.

### Ex08 – String Repetition and Concatenation (ASCII Card)

Build and display a formatted ASCII welcome card using string repetition and concatenation.

### Ex09 – String Concatenation (URL Builder)

Construct a full API URL by concatenating a base domain, path, resource ID, and action.

### Ex10 – String Count (Hashtag Tracker)

Count exact and lowercase occurrences of a hashtag in a social media post.

### Ex11 – String Formatting (Inventory Management)

Calculate remaining stock after a withdrawal and display the result in a formatted message.

### Ex12 – String Formatting (Installment Calculator)

Calculate the total price and monthly installment of a product with simple interest and an administrative fee.

### Ex13 – Arithmetic Operations (Game Score Calculator)

Calculate a player's final score in a game using base points, bonuses, deductions, and an amplification factor.

### Ex14 – Unit Conversion and String Formatting (Temperature Converter)

Convert a temperature from Celsius to Fahrenheit and calculate a Thermal Stability Indicator, displaying both as a formatted string.

### Ex15 – String Analysis (Vowel Percentage)

Calculate the percentage of vowels in a user-provided message relative to the total number of characters.

### Ex16 – String Concatenation (Structured Log Generator)

Build a standardized financial transaction log entry by concatenating seven input fields in a precise format.
