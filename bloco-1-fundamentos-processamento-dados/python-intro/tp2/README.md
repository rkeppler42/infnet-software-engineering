# TP2 – Introdução à Programação com Python

Performance Test 2. Covers string manipulation, formatting, and basic input/output operations.

---

## Exercises

### Ex01 – Multiline Strings
Create and store a multiline file header as a single string variable.

### Ex02 – String Concatenation (SKU Generator)
Build a product SKU by concatenating a category prefix and a serial number.

### Ex03 – String Repetition (Separator Line)
Generate a visual separator line by repeating a character a given number of times.

### Ex04 – String Methods (Swap Case)
Capture user input and display the original message alongside its swapped-case version.

### Ex05 – String Formatting (Name Badge)
Format a full name to uppercase and a badge name to title case, then build a security ID string.

### Ex06 – String Search (Password Validator)
Check if a password contains specific special characters and verify if it is all-lowercase or all-uppercase.

### Ex07 – String Length (SMS Validator)
Calculate the total length of an SMS message with prefix and suffix, and determine how many characters it is over or under the 160-character limit.

### Ex08 – String Repetition and Concatenation (ASCII Card)
Build and display a formatted ASCII welcome card using string repetition and concatenation.

### Ex09 – String Concatenation (URL Builder)
Construct a full API URL by concatenating a base domain, path, resource ID, and action.

### Ex10 – String Count (Hashtag Tracker)
Count exact and lowercase occurrences of a hashtag in a social media post.

### Ex11 – String Formatting (Inventory Management)
Calculate remaining stock after a withdrawal and display the result in a formatted message.

### Ex12 – String Formatting (Installment Calculator)
Calculate the total price and monthly installment of a product with simple interest and an administrative fee.

### Ex13 – Arithmetic Operations (Game Score Calculator)
Calculate a player's final score in a game using base points, bonuses, deductions, and an amplification factor.

### Ex14 – Unit Conversion and String Formatting (Temperature Converter)
Convert a temperature from Celsius to Fahrenheit and calculate a Thermal Stability Indicator, displaying both as a formatted string.

### Ex15 – String Analysis (Vowel Percentage)
Calculate the percentage of vowels in a user-provided message relative to the total number of characters.

### Ex16 – String Concatenation (Structured Log Generator)
Build a standardized financial transaction log entry by concatenating seven input fields in a precise format.

# TP2 – Introdução à Programação com Python

Teste de Performance 2. Cobre manipulação de strings, formatação e operações básicas de entrada e saída.

---

## Exercícios

### Ex01 – Strings Multilinha
Criar e armazenar um cabeçalho de arquivo multilinha em uma única variável string.

### Ex02 – Concatenação de Strings (Gerador de SKU)
Construir um SKU de produto concatenando um prefixo de categoria e um número de série.

### Ex03 – Repetição de Strings (Linha Separadora)
Gerar uma linha separadora visual repetindo um caractere um número determinado de vezes.

### Ex04 – Métodos de String (Swap Case)
Capturar a entrada do usuário e exibir a mensagem original junto com sua versão com capitalização invertida.

### Ex05 – Formatação de Strings (Crachá)
Formatar um nome completo em maiúsculas e um nome de crachá em title case, e construir uma string de identificação de segurança.

### Ex06 – Busca em Strings (Validador de Senha)
Verificar se uma senha contém caracteres especiais específicos e se é composta apenas por letras minúsculas ou maiúsculas.

### Ex07 – Comprimento de String (Validador de SMS)
Calcular o comprimento total de um SMS com prefixo e sufixo, e determinar quantos caracteres estão acima ou abaixo do limite de 160.

### Ex08 – Repetição e Concatenação de Strings (Cartão ASCII)
Construir e exibir um cartão de boas-vindas em ASCII usando repetição e concatenação de strings.

### Ex09 – Concatenação de Strings (Construtor de URL)
Construir uma URL completa de API concatenando domínio base, caminho, ID de recurso e ação.

### Ex10 – Contagem em Strings (Rastreador de Hashtag)
Contar ocorrências exatas e em minúsculas de uma hashtag em uma postagem de mídia social.

### Ex11 – Formatação de Strings (Gestão de Estoque)
Calcular o estoque restante após uma retirada e exibir o resultado em uma mensagem formatada.

### Ex12 – Formatação de Strings (Calculadora de Parcelas)
Calcular o preço total e a parcela mensal de um produto com juros simples e taxa administrativa.

### Ex13 – Operações Aritméticas (Calculadora de Pontuação)
Calcular a pontuação final de um jogador usando pontos base, bônus, deduções e fator de amplificação.

### Ex14 – Conversão de Unidades e Formatação (Conversor de Temperatura)
Converter uma temperatura de Celsius para Fahrenheit e calcular um Indicador de Estabilidade Térmica, exibindo ambos como string formatada.

### Ex15 – Análise de Strings (Porcentagem de Vogais)
Calcular a porcentagem de vogais em uma mensagem fornecida pelo usuário em relação ao total de caracteres.

### Ex16 – Concatenação de Strings (Gerador de Log Estruturado)
Construir um registro de log de transação financeira padronizado concatenando sete campos de entrada em um formato preciso.
