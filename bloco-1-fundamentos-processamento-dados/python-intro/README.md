# 🐍 Introdução à Programação com Python

Foundational Python course covering core programming concepts through hands-on exercises.

> 📝 Exercise files are written in Portuguese, as required by the institution.

## Topics Covered

- Data types, variables, and operators
- Control flow: conditionals and loops
- Functions and documentation
- Lists, strings, and basic data structures
- Algorithms and problem-solving

# 🐍 Introdução à Programação com Python

Disciplina introdutória de Python cobrindo conceitos fundamentais de programação através de exercícios práticos.

> 📝 Os arquivos de exercícios estão escritos em português, conforme exigido pela instituição.

## Tópicos Abordados

- Tipos de dados, variáveis e operadores
- Fluxo de controle: condicionais e loops
- Funções e documentação
- Listas, strings e estruturas de dados básicas
- Algoritmos e resolução de problemas

## Estrutura

| Pasta | Descrição |
|---|---|
| [`tp1/`](./tp1/) | Teste de Performance 1 |
| [`tp2/`](./tp2/) | Teste de Performance 2 |
| [`tp3/`](./tp3/) | Teste de Performance 3 |
| [`at/`](./at/) | Assessment (com nota) — 16 exercícios |

## Destaques do Assessment

- Validação de CPF usando o algoritmo oficial da Receita Federal
- Método de Monte Carlo para estimar π
- Integração numérica (Soma de Riemann)
- Jogo de grid 2D com verificação de fronteiras e game loop
