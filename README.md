
# 🎓 Infnet – Software Engineering

Coursework from the **Software Engineering** program at **Instituto Infnet** (Brazil).

Each folder corresponds to a semester block. Inside each block, courses are organized
with assignments split by evaluation type: `tp1`, `tp2`, `tp3` (mandatory, graded for
feedback only — does not count toward final grade) and `at` (assessment, graded).

Each block introduces different languages and technologies depending on the topic —
ranging from Python and SQL in early blocks to JavaScript, React, Java, C#, and more
in later semesters.
